To use 5.14.2.1-32bit merge module, your Perl::Dist::Strawberry subclass has to use these parameters to new():

  ...
  msm_to_use           => 'http://strawberry-perl.googlecode.com/files/strawberry-perl-5.14.2.1-64bit.msm',
  msm_zip              => 'http://strawberry-perl.googlecode.com/files/strawberry-perl-5.14.2.1-64bit.zip',
  msm_code             => 'BAC35214-5F6A-3E6A-8F42-2F90ADBB1BCF',
  fileid_perl          => 'F_exe_MzA1Mjk2NjIyOQ',
  fileid_relocation_pl => 'F_tab_Mzg0NDgwNzYyMQ',
  ...

For more details see: Using the Strawberry Perl Merge Module
