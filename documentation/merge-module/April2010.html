Merge Module Documentation

Strawberry Perl is now available as a merge module to include in other .msi installers.

To use a merge module in Perl::Dist::Strawberry subclasses:

Your Perl::Dist::Strawberry subclass has to use these parameters to new(), in addition to any others it may need. These parameters require Perl::Dist::WiX version 1.200 or above.

To use 5.8.9.5:

...
msm_to_use           => 'http://strawberryperl.com/download/strawberry-msm/strawberry-perl-5.8.9.5.msm',
msm_zip              => 'http://strawberryperl.com/download/strawberry-perl-5.8.9.5.zip',
msm_code             => 'BC4B680E-4871-31E7-9883-3E2C74EA4F3C',
...

To use 5.10.1.2:

...
msm_to_use           => 'http://strawberryperl.com/download/strawberry-msm/strawberry-perl-5.10.1.2.msm',
msm_zip              => 'http://strawberryperl.com/download/strawberry-perl-5.10.1.2.zip',
msm_code             => 'BC4B680E-4871-31E7-9883-3E2C74EA4F3C',
...

To use the 32-bit version of 5.12.0.1:

...
msm_to_use           => 'http://strawberryperl.com/download/strawberry-msm/strawberry-perl-5.12.0.1.msm',
msm_zip              => 'http://strawberryperl.com/download/strawberry-perl-5.12.0.1.zip',
msm_code             => 'BC4B680E-4871-31E7-9883-3E2C74EA4F3C',
fileid_perl          => 'F_exe_MzA1Mjk2NjIyOQ',
fileid_relocation_pl => 'F_lp_NDIwNjE2MjkyNw',
...

To use the 64-bit version of 5.12.0.1:

...
msm_to_use           => 'http://strawberryperl.com/download/strawberry-msm/strawberry-perl-5.12.0.1-64bit.msm',
msm_zip              => 'http://strawberryperl.com/download/strawberry-perl-5.12.0.1-64bit.zip',
msm_code             => 'BC4B680E-4871-31E7-9883-3E2C74EA4F3C',
fileid_perl          => 'F_exe_MzA1Mjk2NjIyOQ',
fileid_relocation_pl => 'F_lp_NDIwNjE2MjkyNw',
...

In addition, your 'tasklist' parameter has to use 'initialize_using_msm' as its second entry (right after 'final_initialization'.)
