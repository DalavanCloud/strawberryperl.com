To use 5.14.2.1-32bit merge module, your Perl::Dist::WiX subclass has to use these parameters to new():

  ...
  msm_to_use           => 'http://strawberry-perl.googlecode.com/files/strawberry-perl-5.14.2.1-32bit.msm',
  msm_zip              => 'http://strawberry-perl.googlecode.com/files/strawberry-perl-5.14.2.1-32bit.zip',
  msm_code             => '72014D7D-5EC7-3665-9F97-959C988CA754',
  fileid_perl          => 'F_exe_MzA1Mjk2NjIyOQ',
  fileid_relocation_pl => 'F_tab_Mzg0NDgwNzYyMQ',
  ...

For more details see: Using the Strawberry Perl Merge Module
