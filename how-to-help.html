How can I help?

  1. You can help by answering questions that people ask on the #win32 irc channel. Be polite when doing so.

  2. You can help by testing beta versions when they're available.

    If you can do this, http://strawberryperl.com/beta/ is the place to find them. Beta versions will also be announced on the Win32 Perl list.

    We attempt to work on a 3-month period before releases, with the first month being where we add new things, and the second month being where we start the beta testing, so the beta versions for April 2010 will be available in early March 2010.

    Beta versions will be available soon (they may come earlier) for "Strawberry Perl Professional" and 64-bit versions, so just wait a bit.

    You may wish to install CPAN::Reporter on your betas, and report modules that cannot be installed to their authors.

  3. You can help in coding the builder modules.

    In order to do this:

    1. Get SlikSVN or TortoiseSVN or both. Also install a version of "Bootstrap Perl" (which can be downloaded from the beta page above.)

    2. Check out these modules and install them: (you only need a login to check changes back in.)

      http://svn.ali.as/cpan/trunk/WiX (lowest level, provides objects that are equivalent to XML "tags")
      http://svn.ali.as/cpan/trunk/Perl-Dist-WiX (The basic builder software)
      http://svn.ali.as/cpan/trunk/Perl-Dist-Strawberry
      http://svn.ali.as/cpan/trunk/Perl-Dist-Chocolate ("Strawberry Perl Professional")

    3. Read Perl::Dist::WiX::Repository and Perl::Dist::Strawberry::Repository - this explains how to get a login, and what the current goals are.

    Most areas of work on the builder modules require Perl skills. Some areas require C skills, as well.

  4. You can help by providing binaries for libraries that aren't currently in Strawberry:

    E-mail kmx@cpan.org and csjewell@cpan.org. We may wish to provide the library as part of Strawberry Perl or Strawberry Perl Professional, or provide the library as an after-installation .zip. This area requires C skills, of course.
