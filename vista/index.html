Platform Equality Completed!!!

Due to a problem with the bundled MinGW C/C++ Compiler Toolchain Strawberry Perl 5.10.0 and 5.10.0.1 could not install CPAN modules that incorporated C/XS elements on Windows Vista.

This problem was resolved by 5.10.0.2 for all versions of Windows Vista.

Further, Windows Vista Service Pack 1 has provided a fix for the older 5.10.0 and 5.10.0.1 releases of Strawberry Perl.
