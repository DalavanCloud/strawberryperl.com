Strawberry Perl Release Notes

Version 5.12.0-RC0.0 Beta 1, released 25 Mar 2010

Check out what is new, what known issues there are, and frequently asked questions about this version of Strawberry Perl. As always, you're encouraged to tell us what you think.

What's new in Strawberry Perl 5.12.0-RC0.0 Beta 1: Collapse

  • Updated all included dynamic link libraries (except MySQL) with ones that will not conflict with anything else on the system.
    (An update to MySQL, and some other new libraries, will be included in Beta 2.)
  • Updated all included modules with current CPAN versions.
  • Added File::Slurp and a version-checking script.
  • Added the ability to install Strawberry Perl in any directory without spaces.

Known issues: Collapse

  • CPAN will occasionally fail to move the directory that a distribution is temporarily unpacked to. (RT#55924)
  • 64-bit versions of Windows may cause an invalid memory access when uninstalling. (This will be investigated before Beta 2)
  • MySQL is not installed in this version of Perl. (This will be fixed before Beta 2)
