Strawberry Perl Professional Release Notes

Version 5.10.1.3 Alpha 2, released 25 Jul 2010

Check out what is new, what known issues there are, and frequently asked questions about this version of Strawberry Perl Professional. As always, you're encouraged to tell us what you think.

What's new in Strawberry Perl Professional 5.10.1.3 Alpha 2: Collapse

We've added Frozen Bubble, as well as updating Padre to 0.64, Task::Kensho, Task::Catalyst, Task::Moose, and all their dependencies.

We've divided the long list of start menu options into subcategories (see http://csjewell.dreamwidth.org/15458.html for screenshots based on a development snapshot.)

The README.txt file is now on the start menu, as well.


Known issues: Collapse

  • SDL got installed into "site" rather than "vendor".
  • The README file could be improved.
  • Better graphical tools are needed. (Intended for Alpha 3.)
